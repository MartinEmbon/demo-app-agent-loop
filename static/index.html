How it works in code

The whole loop above runs on real AgentLoop endpoints. Here's the Python that powered each step.

Step 1 & 3 — Ask the agent 
# Search relevant past corrections (scoped to this visitor)
memories = loop.search(question, user_id=session_id, tags=["demo"])

# Inject them into the system prompt
system = base_prompt + "\nTrusted facts:\n" + memory_text(memories)

# Ask the LLM. wrapped_openai auto-logs the turn for review.
response = wrapped_openai.chat.completions.create(
    model="gpt-4o-mini",
    messages=[
        {"role": "system", "content": system},
        {"role": "user", "content": question},
    ],
    agentloop={"user_id": session_id, "tags": ["demo"]},
)
Step 2 — Record the correction 
loop.annotate(
    question=question,
    agent_response=wrong_answer,
    correction=user_supplied_correct_answer,
    rating="incorrect",
    root_cause="context",
    user_id=session_id,
    tags=["demo"],
)
# Returns immediately. The annotation is embedded, deduplicated,
# and available to search() within seconds.
